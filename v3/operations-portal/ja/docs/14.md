# 14. 年度ごとの期日設定方法

年度が変わる際に、各申請の期日を設定する必要があります。
ホーム画面の [年度別期日管理] のタブをクリックします。
![[[Screenshot]]](img/14_A.png)

画面右上の [新規] ボタンをクリックします。
作成画面のダイアログが表示されます。
![[[Screenshot]]](img/14_B.png)

「年度」は半角数字4桁で入力して下さい。  
各日付を入力する際は、「YYYY/MM/DD」の形で入力、あるいは入力欄右側のカレンダーのボタンをクリックし
日付を選択して下さい。
[保存] をクリックするとデータが保存されます。
